# Jelium.ai

Jelium.ai is a machine learning-powered web application designed to predict a student's probability of clearing the Joint Entrance Examination (JEE) Advanced cutoff. The prediction engine is trained on 15 years of official National Testing Agency (NTA) historical data, utilizing a logistic regression classifier to analyze the shifting dynamics between Physics, Chemistry, and Mathematics scores.

## Architecture and Technology Stack

The project is structured as a decoupled full-stack application.

### Frontend
* Framework: React 18 (Bootstrapped with Vite)
* Styling: Tailwind CSS
* Icons: Lucide React
* UI/UX: Pure black dark-mode aesthetic with cyan-to-indigo gradient accents, featuring custom CSS animations (floating mathematical equations) and a rapid-fire equation loading sequence.

### Backend
* Framework: FastAPI (Python)
* Machine Learning: Scikit-Learn (Logistic Regression model)
* Data Processing: Pandas, NumPy
* Server: Uvicorn

## Core Features

* Historically Anchored Algorithm: Maps exact raw marks to percentiles spanning over a decade of official cutoffs rather than relying on basic mock formulas.
* Real-time Analytics: A high-performance FastAPI backend ensures instant probability updates as users adjust their expected marks.
* Premium User Interface: A highly polished, responsive frontend featuring interactive range sliders, dynamic progress bars, and an immersive mathematical background.
* Category-Specific Analysis: Incorporates reservation categories (General, EWS, OBC-NCL, SC, ST) into the probability matrix.

## Getting Started

To run Jelium.ai locally, you will need to start both the Python backend server and the React development server concurrently.

### Prerequisites
* Python 3.8 or higher
* Node.js (v16 or higher) and npm

### Backend Setup

1. Open a terminal and navigate to the root directory of the project.
2. (Optional but recommended) Create and activate a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
Install the required Python dependencies:

Bash


pip install fastapi uvicorn scikit-learn pandas pydantic
Start the FastAPI server:

Bash


uvicorn server:app --reload
The backend will now be running at http://127.0.0.1:8000.

Frontend Setup
Open a second terminal tab and navigate to the frontend directory:

Bash


cd aaghaaz-frontend
Install the Node modules:

Bash


npm install
Start the Vite development server:

Bash


npm run dev
Open your browser and navigate to http://localhost:5173 to view the application.

Usage
Navigate to the "Predictor" tab via the top navigation bar or the "Launch Predictor" button.

Select the applicable Reservation Category from the dropdown menu.

Adjust the sliders to input the expected marks (out of 100) for Physics, Chemistry, and Mathematics.

Click "Calculate Probability" to initiate the analysis.

Review the total marks and the computed confidence score.
